# Icon Kits

Save **unlimited icon presets** and instantly switch between them without replacing your current setup.

## How to Use

1. Open the **Garage** in Geometry Dash
2. Click the **"Kits"** button
3. Create, load, and manage your icon presets

## Features

- Save & load unlimited icon kits
- Every cosmetic option: icons, colors, glow, death effect, streak
- Favorite, search, sort, duplicate, rename
- Import & export kits as JSON via clipboard
- Apply a random kit for fun
- Auto-saves all changes
