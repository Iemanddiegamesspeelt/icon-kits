# Changelog

## v1.0.1

- Split bottom buttons in the kits list popup into two rows to prevent clipping on small screen sizes.
- Added copy/duplicate preset feature directly from another player's profile page.
- Fixed a bug where loading a preset would visually change player colors but not immediately update the player frame preview.
- Added Up/Down arrow buttons on each kit cell for manual kit list reordering.

## v1.0.0

- Initial release
- Save and load unlimited icon kits
- Full cosmetic capture (all icon types, colors, glow, effects)
- Create, rename, delete, duplicate kits
- Favorite kits with star toggle
- Search kits by name
- Sort by name, date, or favorites
- Random kit selection
- Import/export via clipboard JSON
- Auto-save on every change
